## Switching katas

1. Pause — last log entry should say **Next:** …
2. Set **Active** above (or tell the agent which kata).
3. Open or create `katas/godot/<id>/learning-log.md` for the new kata.
4. Coming back to an old kata — read **that** file; resume at **Next**.

Branch name can match (`02-under-ceiba-canopy`) but the log follows the **folder**, not the branch — survives merge and branch delete.
