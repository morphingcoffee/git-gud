# Art assets

Engine-agnostic production sources. Import into `katas/godot/`, `katas/unity/`, or `katas/unrealengine/` projects — do not commit engine import caches here.

| Subfolder | Contents |
|-----------|----------|
| `blender/` | `.blend` masters, exports |
| `textures/` | Sprites, UI vectors, channel-packed textures |
| `audio/` | Linear masters, stems |
